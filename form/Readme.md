## Overview
This project is a **modern web form** built using **HTML and CSS** as part of my front-end development learning journey.

## Learning Objectives
This project helped me practice and understand:
- Structuring forms with semantic HTML (`form`, `fieldset`, `legend`, `label`, etc.)  
- Designing accessible and consistent input elements.  
- Implementing interactive states using CSS pseudo-classes (`:focus`, `:valid`, `:placeholder-shown`).  
- Using CSS variables for **color theming** and reusability.  
- Creating a layout with **consistent spacing and shadow effects**.
